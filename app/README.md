# Darzi — Mobile App

Expo / React Native app for **Darzi** (درزی), with support for iOS, Android, and Web.

## Setup

```bash
npm install
npm start
```

- **iOS**: `npm run ios`
- **Android**: `npm run android`
- **Web**: `npm run web`

## Stack

- **Expo** 54
- **React Native** with **NativeWind** (Tailwind CSS)
- **TypeScript**
- **Fonts**: Noto Nastaliq Urdu, DM Sans, Cormorant Garamond

## Folders

| Folder   | Purpose                          |
|----------|----------------------------------|
| [theme/](theme/)   | Design tokens (colors, shadows)  |
| [assets/](assets/) | Icons, images, splash, favicon   |

## Entry

- `App.tsx` — Root component
- `index.ts` — Entry point
- `app.json` — Expo config
