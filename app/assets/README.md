# Assets

Static images and icons for the Darzi app.

## Contents

- **App icon** — `icon.png` (Expo app icon)
- **Splash** — `splash-icon.png`
- **Favicon** — `favicon.png` (web)
- **Brand** — `darzii.png`
- **Android** — `android-icon-*.png` (adaptive icon layers)

## Usage

Reference via Expo asset resolution or direct path, e.g. `require('./assets/icon.png')` or `@/assets/...` if aliased.

Keep sizes and formats per [Expo asset guidelines](https://docs.expo.dev/develop/user-interface/splash-screen-and-app-icon/).
