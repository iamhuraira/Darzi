# Darzi — درزی

**Darzi** (درزی — سلائی اور تیاری) is a tailoring and sewing platform.

## Project structure

| Folder    | Description                    |
|-----------|--------------------------------|
| [app/](app/)       | Expo / React Native mobile app (iOS, Android, Web) |
| [backend/](backend/) | API and server (planned)       |
| [admin/](admin/)   | Admin dashboard (planned)      |

## Getting started

### Mobile app

```bash
cd app
npm install
npm start
```

Then run on a device/simulator: `npm run ios`, `npm run android`, or `npm run web`.

### Tech stack

- **App**: Expo 54, React Native, NativeWind (Tailwind), TypeScript
- **Design**: Custom theme (navy/gold), Urdu (Noto Nastaliq), DMSans, Cormorant Garamond
